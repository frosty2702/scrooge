# NIFTY 50 Historical Data

Cleaned daily historical data for the NIFTY 50 index.

## Details
- Source: Yahoo Finance (via yfinance)
- Time range: 17-Sep-2007 to present
- Frequency: Daily
- Columns:
  - date
  - open
  - high
  - low
  - close
  - adj_close
  - volume

## Notes
- Yahoo Finance provides NIFTY 50 data starting from Sep 2007.
- Volume is 0 as NIFTY 50 is an index.
